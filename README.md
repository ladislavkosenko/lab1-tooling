# Lab 1: Tooling Setup

Цей проєкт демонструє налаштування базового інструментарію для розробки на Go (linting, testing, automation).

## Як запустити проєкт
У проєкті налаштовано Makefile для зручності. Використовуйте наступні команди в терміналі:
- `make fmt` — форматування коду за стандартом.
- `make lint` — запуск статичного аналізатора коду (golangci-lint).
- `make test` — запуск unit-тестів.
- `make build` — компіляція бінарного файлу в директорію /bin.
- `make all` — послідовний запуск усіх вищезгаданих команд.

*Примітка: Прапорець `-race` для тестів у Makefile був тимчасово вилучений через відсутність CGO/GCC в середовищі Windows.*
